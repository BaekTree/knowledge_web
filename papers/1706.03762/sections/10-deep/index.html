Attention Is All You Need
arxiv:1706.03762

Section 10 deeper explanation

Abstract

The abstract introduces the paper’s core technical shift: replacing recurrent neural networks and convolutions with an architecture built entirely on attention. In the translation setting, older encoder-decoder models process tokens sequentially, which limits parallelism and makes training slow. The Transformer instead uses attention mechanisms to let each token directly “look at” other tokens in the input or previously generated output. This matters because the model can capture long-range dependencies without having to pass information through many recurrent steps. The abstract also highlights the main empirical claim: this simpler design is not just faster to train, but also better in translation quality. Several design choices are already foreshadowed here. The model uses self-attention, where a sequence attends to itself, and this is combined with multi-head attention, which lets the network learn different kinds of relationships in parallel. The abstract also mentions parameter-free position representation, which is necessary because removing recurrence eliminates the natural notion of order; the model must be told where tokens appear in the sequence. These decisions are motivated by a tradeoff: reduce architectural complexity while preserving the ability to model sequence structure and context. The reported BLEU scores on WMT 2014 English-German and English-French serve as the paper’s evidence that this tradeoff works. This section sets up the rest of the paper by establishing both the motivation and the benchmark for success. The next sections need to explain how the Transformer is built to realize these claims: how attention is computed, how the encoder and decoder are structured, and how positional information is injected. In other words, the abstract says “what changed” and “why it matters,” while the body of the paper will show the exact mechanism that makes the change work.
