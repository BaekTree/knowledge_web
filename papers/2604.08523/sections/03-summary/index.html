Section 3 summary

Benchmark

  • Defines ClawBench as a real-web benchmark for everyday online tasks, not a sandbox or read-only retrieval setup.
  • Explains the paper’s core measurement question: can agents succeed on live, write-heavy workflows like purchases, reservations, and applications?
  • Claims stronger realism and coverage by using 144 live platforms and comparing agent traces against human ground-truth outcomes.
  • Positions ClawBench against prior web-agent benchmarks along four axes: real-world environment, write-heavy tasks, evaluation reliability, and diagnostic traceability.
  • Sets up the evaluation pipeline architecture: task definition, five-layer agent recording, and automated scoring with the Agentic Evaluator.
