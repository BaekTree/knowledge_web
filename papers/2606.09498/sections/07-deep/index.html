5 Weakness MiningProceed to Next

This section describes the self-improvement loop used by Self-Harness: first mine weaknesses from failed runs, then propose candidate harness edits, then validate those edits before promotion. The weakness-mining step groups failures using a deterministic signature built from three pieces: the verifier’s final rejection cause, the causal role of the agent behavior, and the reusable mechanism exposed by the trace. In other words, it does not cluster traces by vague similarity; it clusters only when they point to the same actionable weakness, such as missing validation, unconstrained exploration, or corrupted tool calls. The result is an evidence bundle that summarizes recurring failure patterns without directly telling the model how to patch them. The proposal stage then uses the same fixed model in a new role: instead of solving tasks, it acts as a proposer that reads the evidence bundle plus the editable parts of the current harness. It generates several mutually distinct candidate edits in parallel. This design balances exploration and discipline: diversity helps avoid local fixes that only address one symptom, while minimality ensures each candidate changes only the smallest relevant harness surface. The paper also filters by “addressability,” meaning a failure pattern is only targeted if a harness edit can plausibly affect it; this avoids wasting edits on issues that are really model-capability limits or task ambiguity. The validation stage is a conservative gate. Each candidate harness is evaluated on both held-in data, which checks whether it fixes the observed weakness, and held-out data, which checks for regressions. A candidate is accepted only if it improves at least one split without hurting the other, and stochastic runs are repeated to reduce luck-based promotions. This connects directly to the next step in the paper: only validated edits become the updated harness, which then feeds back into the next mining round, creating an auditable improvement cycle rather than an unconstrained search loop.
