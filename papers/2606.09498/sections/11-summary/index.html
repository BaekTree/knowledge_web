Section 11 summary

Benchmarks

This section defines the evaluation setup for Self-Harness on a controlled subset of Terminal-Bench-2.0 tasks. It sets up the paper’s main test of whether the harness can improve itself by using a benchmark that stresses tool use, verification, and recovery. The key claim is that the authors evaluate on 64 stable cases to reduce noise and avoid unsupported multimodal or web-dependent tasks.
